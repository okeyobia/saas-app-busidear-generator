
# AI Business Idea Generator (Next.js + FastAPI)

This project is a SaaS starter combining a [Next.js](https://nextjs.org) frontend with a [FastAPI](https://fastapi.tiangolo.com/) backend. It generates new business ideas for AI agents using the OpenAI API.

## Features

- **Frontend:** Next.js (React, TypeScript, Tailwind CSS)
- **Backend:** FastAPI (Python)
- **AI Integration:** Calls OpenAI API to generate business ideas

---

## Getting Started


## Getting Started

### 1. Install dependencies

#### Frontend (Next.js)
```bash
npm install
# or yarn install
```

#### Backend (FastAPI)
```bash
cd api
pip install -r requirements.txt
```

### 2. Run the development servers

#### Start FastAPI backend
```bash
uvicorn index:app --reload --port 8000
```

#### Start Next.js frontend
```bash
npm run dev
# or yarn dev
```

### 3. Usage

Open [http://localhost:3000](http://localhost:3000) to use the app. The homepage will display a new AI business idea generated by the backend.

---


## Project Structure

- `pages/` — Next.js frontend (main UI in `index.tsx`)
- `api/` — FastAPI backend (`index.py`)
- `styles/` — Tailwind CSS and global styles

## Backend API

- **Endpoint:** `GET /api`
- **Description:** Returns a new business idea for AI agents (uses OpenAI API)

## Environment Variables

You must set your OpenAI API key for the backend. Create a `.env` file in the `api/` folder:

```
OPENAI_API_KEY=your-key-here
```
